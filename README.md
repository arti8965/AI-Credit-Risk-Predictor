# 🛡️ AI-Powered Credit Risk Assessment Dashboard

An interactive end-to-end Machine Learning web application designed to evaluate loan applicant risk profiles using financial parameters. This project features a professional banking UI and a robust AI backend logic.

## 🚀 Project Overview
This system helps financial institutions automate the initial screening of loan applications. By analyzing key factors like Credit Score, Income-to-Loan ratio, and Employment history, the AI model classifies applicants into **Low**, **Medium**, or **High Risk** categories.

## ✨ Key Features
- **Professional Dashboard:** Built with Streamlit for a clean, corporate look.
- **Real-time AI Analysis:** Dynamic risk calculation based on user input.
- **Interactive Visualizations:** Integrated Plotly charts for data-driven insights.
- **Modular Architecture:** Separated Frontend (`app.py`) and Backend Logic (`model_logic.py`).

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python-based Web Framework)
- **Backend:** Python (Data Processing & Logic)
- **Visualizations:** Plotly Express
- **Deployment:** Streamlit Cloud / GitHub

## 📊 Model Metrics (Mock Data for Interview)
- **Accuracy:** 94.8%
- **Model Type:** Random Forest Classifier (Heuristic Approach)
- **Precision:** 92.4%
